# Amazon-Clone 🛒
My first HTML-CSS project 

A fully responsive front-end clone of the Amazon homepage built from scratch using HTML5 and CSS3. This project focuses on pixel-perfect replication, clean semantic layout, and modern CSS styling techniques.

## ✨ Features
* **Pixel-Perfect Header:** Complete with the Amazon logo, delivery location selector, search bar, language/currency selector, and account/orders dropdown layout.
* **Responsive Multi-Card Product Grid:** Styled product categories using modern layouts that adapt seamlessly to different screen dimensions.
* **Interactive Hover States:** Smooth CSS transitions on links, buttons, and navigation items to mimic the authentic Amazon user experience.
* **Back to Top Button:** A functional visual placeholder for the footer navigation.
* **Semantic HTML:** Structured using proper semantic tags (`<header>`, `<nav>`, `<main>`, `<footer>`, `<section>`) for better accessibility and SEO practices.

---

## 🛠️ Tech Stack
* **HTML5:** Structured the content and layout elements.
* **CSS3:** Flexbox, Grid layout, custom positioning, media queries, and styling.
* **Font Awesome / Remix Icon:** For high-quality, scalable vector icons.

Building this project helped me solidify several core front-end concepts:
1.  **Mastering Flexbox & Grid:** Successfully aligned complex UI components like the massive Amazon search bar and the 4-column product grid layout.
2.  **Component Hierarchy:** Learned how to nest HTML elements efficiently to keep the stylesheet manageable.
3.  **Attention to Detail:** Replicating a massive e-commerce UI taught me the value of precise padding, margin, and typography scales.
